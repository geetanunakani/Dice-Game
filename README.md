
# 🎲 Dicee Game

A fun and simple dice game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Two-player dice game
- Random dice roll on refresh
- Displays the winner dynamically
- Clean and simple UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🎮 How to Play
1. Open the game in your browser
2. Refresh the page
3. Dice will roll automatically
4. Player with higher number wins
